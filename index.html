Exemplo de parágrafo para descrição pessoal

Esta página foi criada com base no aprendizado da Imersão CSS realizada pela Alura .Nesta semana foi possível entender conceitos desde o básico ao avançado sobre a construção de sites utilizando o HTML(linguagem para estruturação de páginas web) e como embelezar esse site através do CSS(linguagem para estilização dos elementos de uma página web).

Dessa forma foi possível ver a importância do CSS, não somente para deixar o site "bonito" mas também para deixá-lo responsivo, e melhorar a experiência de navegação do usuário, o que pode fazer toda a diferença no seu projeto final.

Não foram abordados tópicos como o JavaScript pois não era o foco do evento. Ao final também foi possível ver que os frameworks são muito importantes para a produtividade pois criar o CSS de uma página do zero é algo que vai dar muito trabalho e vai consumir tempo, sendo que muitas configurações podem ser reaproveitadas.

Assim, o conteúdo da semana foi de grande importância. Com o conhecimento adquirido pode-se criar uma página pessoal (como esta) ou também páginas simples para expor um produto ou uma ideia.
